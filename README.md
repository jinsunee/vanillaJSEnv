# VanillaJS environment template for building Web project

## How to install

1. clone this project on your local

```
$ git clone https://github.com/jinsunee/vanillajs-env.git && cd vanillajs-env
```

2. install packages

```
$ yarn
or
$$ npm install
```

## How to start

```
$ yarn
or
$ npm start
```
